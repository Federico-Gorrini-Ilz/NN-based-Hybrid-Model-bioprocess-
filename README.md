NN-Based Hybrid Model is a physics-informed neural network framework for learning and simulating nonlinear process dynamics from noisy data. The project combines a mechanistic state-space model with two neural networks that estimate latent growth and uptake terms, then integrates the full system with RK4 to predict the evolution of key process states over time. It includes synthetic data generation with process and measurement noise, batch-based training, iterative initial-condition identification, and support for both fixed and varying input intervals. Built with PyTorch, SciPy, NumPy, and Matplotlib, the notebook is designed for hybrid modeling, system identification, and robust dynamic prediction in bioprocess-style applications.
